VisIEon - Beginner Instructions to Run Locally
=============================================

Follow these simple steps to run the VisIEon application on your computer. No technical background needed!

1. Make sure all files are together:
   - You should see a file called "index.html" and a folder called "tm-my-image-model" in the same place.

2. Download and install Python (if you don't have it):
   - Go to https://www.python.org/downloads/
   - Click "Download Python" and install it (just click "Next" until it's done).

3. Open the folder on your computer:
   - Find the folder with "index.html" and "tm-my-image-model".
   - Click in the folder's address bar and type "cmd" then press Enter. This opens a black window.

4. Start the local server:
   - In the black window, type this and press Enter:
     python -m http.server

5. Open the app in your browser:
   - After you run the command, open Google Chrome (or Edge).
   - Type this in the address bar:
     http://localhost:8000
   - Press Enter.

6. Allow camera access:
   - If the browser asks for camera permission, click "Allow".

7. Use the app!
   - You will see the VisIEon app. Click "Start Detection" to begin.
   - You can see results and debug logs below the camera.

If you have any trouble, just close everything and try again from step 3.

You do NOT need to install anything else. Just follow these steps and it will work!

For help, ask your team or supervisor.
